Hospedagem GitHub

Para criar um site no GitHub.com devemos ter uma conta valida e um repositorio com um arquivo "index.html"

Hospedando um site

  1. Criar um repositorio;
  2. Publicar o repositorio utilizando o publish;
  3. Abrir o repositorio no Vs Code;
  4. Criar o index.html;
  5. Criar a versao usando o commit;
  6. Enviar para o repositorio remoto usando o push;
  7. Entrar no GitHub.com, acessar o repositorio -> Settings -> Pages -> Escolher a branch do source;
  8. Clicar na URL gerada;

Branches

Sao ramificacoes do seu projeto. O projeto principal em geral, fica na branch main. A branch main deve ter a versao final e funcional do seu projeto. O projeto pode ter branches secundarias.
